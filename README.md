# MacGPT
> [!IMPORTANT] 
> This is a work-in-progress project. Chances are it will not be regularly maintained, and will only gain updates when I have the time, or when it stops functioning.

This is a functional visual application that is designed to interact with the built-in macOS 26+ LLM on all apple silcon computers. It utilizes the repository [apfel](https://github.com/Arthur-Ficial/apfel) to access macOS's built-in offline LLM included in macOS 26 and above.
